# 捐款分析儀表板

一個純前端的互動式視覺化工具，協助非營利組織從 LINE/街口 匯出的捐款 CSV 資料中，快速產生完整的捐款分析報告。無需伺服器、無需安裝，直接在瀏覽器中使用。

---

## 功能特色

- **零安裝**：單一 HTML 檔案，直接用瀏覽器開啟即可
- **隱私安全**：所有資料在瀏覽器本機處理，不上傳至任何伺服器
- **互動式圖表**：月度趨勢、方案分佈、性別統計、金額區間分析等多維度圖表
- **手動分類**：可根據組織需求，手動將專案名稱分類
- **重複捐款人識別**：自動標記曾多次捐款的支持者
- **彈性時間篩選**：可依年份或自訂日期範圍篩選資料
- **一鍵匯出**：支援將分析結果匯出為 HTML 報告（即本專案中的「儀表板分析結果」範例）

---

## 如何使用

**1. 下載儀表板**

下載 `LINE捐款分析儀表板_分類.html` 並儲存至本地端。

**2. 準備資料**

從 LINE 購物後台匯出捐款訂單，確認 CSV 欄位包含：

| 欄位名稱 | 說明 |
|---|---|
| `性別` | 1＝男、2＝女、空白＝未填 |
| `商品名稱` | 捐款方案名稱（含 emoji 亦可） |
| `總數量` | 購買數量（選填） |
| `商品金額` | 單價（選填） |
| `總金額` | 該筆捐款總金額 |
| `訂購日` | 格式：`YYYY-MM-DD HH:mm:ss` |
| `捐款人` | 捐款者姓名 |
| `電子郵件` | 電子郵件(選填) |

**3. 開啟儀表板並載入資料**

用任何現代瀏覽器（Chrome、Firefox、Edge、Safari）開啟 HTML 檔，點選「上傳 CSV」按鈕，選取您的捐款資料即可自動產生分析圖表。

**4. 查看完整結果範例**

可直接用瀏覽器開啟預覽儀表板的呈現效果。

## 自訂分類規則

儀表板會依商品名稱中的關鍵字，將捐款自動歸入不同方案類別。若您的組織使用不同的活動名稱，請用文字編輯器開啟 `捐款分析儀表板_分類.html`，搜尋 `CATEGORY_RULES`（或類似的分類設定區塊），依照現有範例新增或修改關鍵字對應即可。

## 適用場景

- 環保、社福、教育等 NPO / NGO 定期整理 LINE 捐款報告
- 籌款活動結束後快速產製成果摘要
- 向董事會或捐款人呈現募款成效

## 系統需求

- 無網路連線需求（資料處理完全在本機進行）
